# Ad Blocker for YouTube & Twitch

![Ad Blocker popup](popup.png)

Ad Blocker removes pre-roll and mid-roll ads on YouTube and Twitch and cleans up banners and overlays so videos stay uninterrupted. All filtering happens locally in the browser without sending viewing data to external servers.

## Highlights

- Skip pre-roll and mid-roll ads on YouTube and Twitch
- Remove overlays, banners, and sponsored elements
- Handle Twitch SSAI ad segments
- Lightweight and fast
